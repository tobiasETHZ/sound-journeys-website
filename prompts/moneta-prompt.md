PROJECT SUMMARY

Build a website for Sound Journeys, a curated immersive listening experience project based in Zurich.

This is NOT:
- a generic event page
- a concert promo site
- a sound bath wellness template
- a therapy or healing website
- a mystical/spiritual retreat brand

This IS:
- an artistic, inward-facing, curated listening format
- a structured, emotionally nuanced group experience
- a container for deep attention, strong musical arcs, and participant autonomy
- a project co-created by PA / Baribahl and Tobias

The site should feel:
- beautiful
- intriguing
- trustworthy
- emotionally intelligent
- refined
- locally anchored in Zurich, but not provincial
- atmospheric without becoming overly cosmic or abstract
- nocturnal with grounded tactile cues
- polished, but not corporate

---

CORE CONCEPT

Sound Journeys are immersive listening sessions in Zurich.

They involve:
- around 90–120 minutes of continuous listening
- a total guest experience of roughly 2 to 3 hours including arrival and integration
- an intentional arc
- inward-facing attention
- the possibility of emotional intensity
- autonomy, consent, and psychological safety
- a format that is artistic rather than therapeutic

Therapeutic or cathartic effects may happen as side effects, but the project should NOT be framed as therapy, healing, treatment, coaching, or spiritual instruction.

The website should help newcomers understand:
- what a Sound Journey is
- what the experience feels like
- what makes it different
- what to expect
- who is behind it
- what upcoming events exist
- how to inquire about private bookings

- arriving / settling / orientation
- grounding / attunement / soft opening
- texture building / tension / immersion

- intensity apex or peaks
- release / comedown / spaciousness
- silence / aftercare / integration

Copy should lead through:
attention → intensity → release

The anatomy should reflect:
- emotional arc
- sensory arc
- physical/embodied arc

Build a signature visual component here.
